# SO Optimizer

**A better one click created by Sinz**

SO Optimizer is a powerful, safe, and research-backed Windows optimization suite designed to maximize performance, reduce latency, and debloat your system without compromising stability.

## Features

*   **System Optimization**: Deep registry tweaks for responsiveness and network performance.
*   **Debloating**: Removes telemetry, tracking services, and unnecessary bloatware.
*   **Gaming Compatibility**: Specifically tuned to keep Minecraft (Java & Bedrock) and Rockstar Games working perfectly.
*   **Privacy**: Enhances user privacy by disabling data collection.
*   **Visuals**: Includes custom SO Wallpaper and visual tweaks.
*   **External Tools**: Integrated support for specialized batch tools.

## Included Tools (SO Folder)

The `SO Folder` contains specialized scripts and assets:

*   **[SO] Task Destroyer**: Safely disables telemetry tasks (Xbox compatible).
*   **[SO] Process Destroyer**: Terminates background bloatware (OneDrive, Cortana, etc.).
*   **[SO] Update Disabler**: Controls Windows Update services.
*   **[SO] Edge Remover**: Script to uninstall Microsoft Edge (requires setup.exe).
*   **SO Profile.nip**: Custom Nvidia Profile Inspector settings.
*   **SO Wallpaper.png**: The official wallpaper.

## Usage

1.  Download the latest release.
2.  Right-click `SO Optimizer.ps1` and select **Run with PowerShell**.
3.  Follow the on-screen menu to apply optimizations.
    *   **Option [1]**: Run the full SO Optimization suite (Recommended).
4.  Restart your computer to apply all changes.
