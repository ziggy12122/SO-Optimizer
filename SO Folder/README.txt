SO Optimizer Tool Suite
=======================

This folder contains the external tools required for the SO Optimizer.

MISSING BINARIES:
-----------------
The following tools cannot be generated and must be placed here manually:

1. NSudo (NSudo.exe) - For running commands with TrustedInstaller privileges.
2. O&O ShutUp10 (OOSU10.exe) - For telemetry disabling.
3. Nvidia Profile Inspector (nvidiaProfileInspector.exe) - For GPU profile management.
4. Edge Setup (setup.exe) - Place the Microsoft Edge Setup executable here for the Edge Remover script.

INCLUDED SCRIPTS:
-----------------
- Task Destroyer.bat: Removes scheduled tasks.
- Process Destroyer.bat: Stops unwanted processes.
- Update Disabler.bat: Disables Windows Update services.
- Edge Remover.bat: Uninstalls Edge (requires setup.exe).

Place your custom .pow power plan files in this folder as well.
